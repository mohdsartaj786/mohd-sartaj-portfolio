## 📩 Contact Form Setup (EmailJS)

1. Create an account on EmailJS
2. Add Email Service (Gmail etc.)
3. Create Email Template
4. Replace these in your code:

```js
emailjs.init("service_fa1ltrt");

emailjs.sendForm(
  "service_fa1ltrt",
  "template_u20dllh",
  form
);
```
